# license-check

Recursively checks the license for a PyPi package.

Usage:

```sh
$ git clone https://github.com/zachriggle/license_check
$ pip install -e license_check
$ license_check pwntools
       capstone@3.0.4    -- UNKNOWN
           cffi@1.8.2    -- MIT
   cryptography@1.5      -- BSD or Apache License, Version 2.0
         enum34@1.1.6    -- BSD License

           Mako@1.0.4    -- MIT
     MarkupSafe@0.23     -- BSD
       paramiko@2.0.2    -- LGPL
         pluggy@0.3.1    -- MIT license
         psutil@4.3.1    -- BSD

             py@1.4.31   -- MIT license
         pyasn1@0.1.9    -- BSD
      pycparser@2.14     -- BSD
     pyelftools@0.24     -- Public domain

       pyserial@3.1.1    -- BSD

```
